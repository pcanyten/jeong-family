# 터미널로 로컬 실행하기 (초보자용)

Node 설치는 했는데, "명령어를 어디서 어떻게 치는지" 모르시는 분을 위한 가이드입니다.

---

## 1. "터미널"이 뭔가요?

**터미널** = 검은(또는 흰) 창에 글자만 나오고, 여기에 **명령어를 입력**해서 컴퓨터에게 일을 시키는 곳입니다.

- **Mac**: "터미널" 앱 (Spotlight에서 "터미널" 검색해서 실행)
- **Windows**: "명령 프롬프트" 또는 "PowerShell" (시작 메뉴에서 검색)
- **Cursor/VS Code 사용 중이면**: 상단 메뉴 **터미널(Terminal)** → **새 터미널(New Terminal)** 로 열어도 됩니다.

---

## 2. 프로젝트 폴더가 어디 있는지

이 프로젝트는 보통 아래 경로에 있습니다.

- **Mac**: `Desktop(바탕화면) > 개인 프로젝트 > jeong-family`
- **전체 경로**: `/Users/pcany/Desktop/개인 프로젝트/jeong-family`

즉, **정씨네 이야기(jeong-family) 프로젝트가 들어 있는 폴더**가 "프로젝트 폴더"입니다.  
그 안에 `package.json`, `src` 폴더 등이 보여야 합니다.

---

## 3. 순서대로 할 일 (복사해서 쓰세요)

### 3-1. 터미널 열기

Mac이면 **Spotlight(⌘+Space)** 에 "터미널" 입력 후 실행.  
또는 Cursor에서 **터미널 → 새 터미널** 로 열기.

---

### 3-2. 프로젝트 폴더로 이동

터미널에 **아래 한 줄 전체**를 복사해서 붙여넣고 **Enter** 치세요.

```bash
cd "/Users/pcany/Desktop/개인 프로젝트/jeong-family"
```

- `cd` = "이 폴더로 이동해라" 는 뜻입니다.
- 경로에 한글(개인 프로젝트)이 있어서 **큰따옴표**로 감싼 겁니다. 꼭 그대로 복사해서 쓰세요.

**성공했으면** 터미널 맨 앞에 `jeong-family` 같은 이름이 보이거나, 다음 명령어를 쳐도 "No such file" 같은 에러가 나지 않습니다.

---

### 3-3. 패키지 설치 (처음 한 번만)

같은 터미널에서 **아래 한 줄** 입력 후 **Enter**:

```bash
npm install
```

몇십 초 정도 걸릴 수 있습니다. 끝나면 다시 입력할 수 있는 상태로 돌아옵니다.

---

### 3-4. 개발 서버 실행

그 다음 **아래 한 줄** 입력 후 **Enter**:

```bash
npm run dev
```

- "Local: http://localhost:3000" 같은 문구가 나오면 **성공**입니다.
- 이 터미널 창은 **끄지 마세요**. 켜 둔 채로 브라우저로 가야 합니다.

---

### 3-5. 브라우저에서 열기

1. **웹 브라우저**(Chrome, Safari 등)를 연다.
2. 주소창에 **아래 주소**를 입력하고 Enter.
   ```
   http://localhost:3000
   ```
3. **정씨네 이야기** 로그인/가입 화면이 보이면 로컬 실행이 완료된 겁니다.

---

### 3-6. 서버 끄는 방법

- 터미널에서 **Ctrl + C** (Mac은 Control + C) 를 한 번 누르면 `npm run dev` 가 멈춥니다.
- 다시 실행하려면 같은 폴더에서 `npm run dev` 를 다시 치면 됩니다.

---

## 4. "프로젝트 폴더가 다른 곳에 있는" 경우

만약 `jeong-family` 를 다른 위치에 두었다면:

1. **Finder(찾기)** 에서 `jeong-family` 폴더를 찾는다.
2. 그 **폴더를 우클릭** → **정보 가져오기** (또는 "경로 복사") 로 전체 경로를 확인한다.
3. 터미널에서는 **항상** 아래처럼 `cd "경로"` 로 그 폴더로 이동한 뒤에 `npm install`, `npm run dev` 를 쳐야 합니다.

예시 (경로만 본인 폴더에 맞게 바꾸기):

```bash
cd "여기에_본인_jeong-family_폴더_전체_경로"
npm install
npm run dev
```

---

## 5. 한 번에 복사해서 쓸 수 있는 명령어 (요약)

**아래 세 줄을 차례대로** 터미널에 붙여넣고 Enter 누르면 됩니다.  
(프로젝트가 `/Users/pcany/Desktop/개인 프로젝트/jeong-family` 에 있다고 가정)

```bash
cd "/Users/pcany/Desktop/개인 프로젝트/jeong-family"
npm install
npm run dev
```

1번째 줄: 프로젝트 폴더로 이동  
2번째 줄: 패키지 설치 (처음 한 번)  
3번째 줄: 서버 켜기 → 브라우저에서 http://localhost:3000 접속

---

네, **터미널을 열고 저 명령어를 직접 입력(또는 복사해서 붙여넣기)** 하시면 됩니다.
